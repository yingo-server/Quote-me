一言 API · 语录服务

基于静态语料库，提供随机或指定条目的 JSON / 纯文本 接口。

基础信息

GET /api/quote

所有请求均支持 CORS，可跨域调用。

请求参数

参数说明可选值默认
type返回格式json, textjson
line指定行号（从 0 开始）整数随机
mode仅 type=text 时生效zh（中文）
en（英文）
both（原始混合）		both

示例请求

GET /api/quote

返回 JSON（随机行）

GET /api/quote?type=text&mode=zh

返回纯文本（仅中文）

GET /api/quote?line=5&type=json

指定行号 JSON

GET /api/quote?type=text&line=42&mode=en

指定行号纯英文

JSON 响应格式

{ "code": 200, "line": 5, "original": "人对人是狼。 Homo homini lupus.", "zh": "人对人是狼。", "en": "Homo homini lupus.", "time": "2026-07-01T10:30:00Z" }

错误处理

  • 行号越界 → 400 错误，提示有效范围
  • 行号非数字 → 自动降级为随机（不报错）
  • 数据文件缺失 → 500 错误
  • 未知 type/mode → 使用默认值
